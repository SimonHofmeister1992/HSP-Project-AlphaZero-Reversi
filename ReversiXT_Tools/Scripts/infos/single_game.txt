Das Script single_game frägt nach dem Namen einer Spieldatei, die sich im Ordner ReversiXT_Tools/Maps/actualLearningMaps/ befindet
um damit den Server sowie 2 weitere Alpha-Zero spieler zu starten (mittels neuer Kommentierung im Script kann auch eine triviale AI
zugeschalten werden).

Das Script single_game_default_map ist identisch aufgebaut, startet jedoch standardmäßig mit der Karte lmap1_2p.map.

Eine Übergabe der Parameter "Zeit" und "Suchtiefe" ist derzeit nicht integriert, die Zeit wird somit automatisch auf Integer.MAX Sekunden 
festgelegt.



Fehlerquellen:

1) Das Konsolenfenster darf nicht vor Ende des Spiels geschlossen werden. Das Ende des Spiels ist derzeit leider
nur über die log_game_*.txt ersichtlich, die sich im selben Ordner befindet, wie das Startskript.


Log-Dateien:
log_game*.txt: Enthält die wichtigsten Spielinformationen
/logs/client*.out entspricht dem Standard-Output des Clients
/logs/server.out entspricht dem Standard-Output des Servers
